# MyTodo

카테고리별로 할 일을 관리할 수 있는 간단한 웹 기반 Todo List 애플리케이션입니다.
순수 HTML/CSS/JavaScript로 작성되었으며, 로그인 후에만 할 일 목록에 접근할 수 있습니다.
계정 정보와 할 일 데이터는 모두 브라우저의 `localStorage`에 저장됩니다.

## 주요 기능

- 로그인 / 회원가입 (로그인 전에는 할 일 목록이 보이지 않음)
- 사용자별로 분리된 할 일 목록
- 할 일 추가 / 삭제 / 수정
- 완료 여부 체크
- 카테고리 분류 (개인, 공부, 업무, 취미)
- 카테고리별 필터링
- 완료율 진행 표시줄(progress bar)
- `localStorage`를 이용한 데이터 유지 (새로고침해도 로그인 상태와 목록 유지)
- 반응형 레이아웃 (모바일 화면 대응)

## 로그인 / 회원가입

- 계정 정보는 `localStorage`의 `mytodo_users`에, 로그인 세션은 `mytodo_session`에 저장됩니다.
- 할 일 목록은 사용자별로 `mytodo_items_<아이디>` 키에 분리되어 저장됩니다.
- 아직 Supabase 연동 전 단계로, 비밀번호가 암호화 없이 브라우저에 그대로 저장되는 프로토타입입니다. 실사용에는 적합하지 않으며, 추후 Supabase Auth 연동 시 대체될 예정입니다.

## 사용 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저로 엽니다.
3. 회원가입 후 로그인하면 할 일 목록 화면으로 전환됩니다.
4. 입력창에 할 일을 작성하고 카테고리를 선택한 뒤 `추가` 버튼을 누르거나 Enter 키를 입력합니다.
5. 체크박스로 완료 처리, ✏️ 버튼으로 수정, 🗑 버튼으로 삭제할 수 있습니다.
6. 상단 필터 탭으로 카테고리별 할 일만 모아 볼 수 있습니다.
7. 우측 상단 `로그아웃` 버튼으로 로그아웃할 수 있습니다.

## 파일 구성

```
.
├── index.html   # 페이지 구조 (로그인/회원가입 + 할 일 목록)

├── script.js    # 동작 로직 (인증 + localStorage 연동)
└── README.md
```

## 기술 스택

- HTML5
- CSS3
